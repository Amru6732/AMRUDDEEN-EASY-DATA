# AMRUDDEEN EASY DATA

## Overview
AMRUDDEEN EASY DATA is a comprehensive data selling platform designed to facilitate smooth transactions and management of data for various use cases. The platform aims to empower businesses with easy access to data to drive informed decision-making processes.

## Features
- User-friendly interface for navigating through data listings.
- Secure payment gateway integration for seamless transactions.
- Ability to search and filter data sets based on various criteria.
- Admin dashboard for managing users, data sets, and transactions.
- Data preview options to review data before purchase.

## Tech Stack
- **Frontend:** React.js
- **Backend:** Node.js, Express
- **Database:** MongoDB
- **Hosting:** AWS

## Installation
1. Clone the repository:
   ```bash
   git clone https://github.com/Amru6732/AMRUDDEEN-EASY-DATA.git
   ```
2. Navigate to the project directory:
   ```bash
   cd AMRUDDEEN-EASY-DATA
   ```
3. Install dependencies:
   ```bash
   npm install
   ```
4. Start the application:
   ```bash
   npm start
   ```
